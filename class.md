## विधि (class)

>_transliteration: vidhi, meaning: set of instruction_

TBD

```ved
विधि रोहन {
  रोहन_में() {
    वद("रोहन में");
  }
}

विधि मोहन सन्धि रोहन {
  मोहन_में() {
    वद("मोहन में");
  }
}

विधि सोहन सन्धि मोहन {
  सोहन_में() {
    वद("सोहन में");
  }
}

मान सोहन_वस्तु = सोहन();
सोहन_वस्तु.रोहन_में();
सोहन_वस्तु.मोहन_में();
सोहन_वस्तु.सोहन_में();
```

**output**

```bash
रोहन में
मोहन में
सोहन में
```
