### Other Linux distributions

On other Linux systems, you can download binary archive from [here](/) and Install via following command:

```bash
mkdir -p $HOME/vedic && tar zxf vedic.tar.gz -C $HOME/vedic
export PATH=$PATH:$HOME/vedic
```
